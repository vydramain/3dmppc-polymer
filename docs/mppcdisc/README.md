# Disc design docs

Design documentation for the games (**discs**) that run on the
[3dmppc console](../platform/). One subdirectory per disc. This mirrors the
buildable content library at [`../../mppcdiscs/`](../../mppcdiscs/): design here,
content there.

## Discs

| Disc                   | Design docs                          | Content                                   |
| ---------------------- | ------------------------------------ | ----------------------------------------- |
| [`solid/`](solid/)     | *Solidmaid: Alkoldun Vasiliusavich* — the reference game. | [`../../mppcdiscs/solid/`](../../mppcdiscs/solid/) |

Start a new disc by adding a `<disc-id>/` folder here with its own `README.md`
and design docs, and a matching `<disc-id>/` under
[`../../mppcdiscs/`](../../mppcdiscs/) for its assets/scripts/data.
